## 🙏 그라운드 룰

- 지각 관리
  - 지각하면 기술 공유하기 (5-10분)
  - 모이기로 하는 시간 기준 +3분 초과 시 지각처리 (3분 1초 금지)
  - 불참하게 된다면 전날에 말해주기

- 빠른 소통을 위한 룰
  - 슬랙 알람 24시간 ON
  - 코어타임 내에는 허들 ON

- 욕설 금지  

- 코어타임 지키기
  - 10 ~ 19 사이 연락이 안되면 ㅎㅎ;;
  - 가능한 19시 이후까지도

- 공식적인 미팅(일정)은 줌으로
  - 팀빌딩
  - 데일리 스크럼

- 비공식적인 미팅은 허들
  - 모각코 포함

- 사랑의 매 피하지 않기 피하면 다시 때리기 
  - 피드백 받고 모르쇠 금지
